# Учёт часов

PWA для учёта рабочих часов и километров. Данные хранятся в браузере телефона.

## Локально

```bash
npm install
npm run dev
```

Открыть на телефоне через локальный IP, который покажет Vite.

## Деплой

При push в `main` GitHub Actions собирает и публикует на GitHub Pages
(см. `.github/workflows/deploy.yml`). Workflow выставляет `BASE_PATH` =
`/<имя-репо>/`, так что приложение доступно по
`https://<логин>.github.io/<имя-репо>/`.

## Установка на iPhone

Открыть URL в Safari → Share → «На экран Домой».

## Бэкап

В нижнем меню списка работников: «Скачать backup» сохраняет JSON со всеми
данными. «Импорт» — восстанавливает из такого файла.
